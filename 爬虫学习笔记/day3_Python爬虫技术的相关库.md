# 三、Python爬虫技术的相关库

## 网络请求

> * `urllib`
> * `requests`/`urllib3`
> * `selenium`(UI自动测试、动态js渲染)
> * `appium`（手机App的爬虫或UI测试）

## 数据解析

> * `re`正则表达式
> * `xpath`
> * `bs4`
> * `json`

## 数据存储

> * `pymysql`
> * `mongodb`
> * `elasticsearch`

## 多任务库

> * 多线程(`threading`)、线程队列`queue`
>
>   * eg:启动一个程序就是一个进程
> * 协程(`asynio`、`gevent`/`eventlet`)

## 爬虫框架

> * `scrapy`
> * `scrapy-redis`分布式（多机爬虫）
